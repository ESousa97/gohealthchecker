# Security Policy

## Reporting a Vulnerability

We take the security of `gohealthchecker` seriously. If you discover a security vulnerability, please do not open a public issue. Instead, send an email to [your-email@example.com].

## Response Process

After reporting, you will receive a response within 48 hours confirming receipt. We will keep you informed of the fix progress and request your validation before publishing the fix.

## Supported Versions

Only the latest stable version receives security updates.

| Version | Supported |
| --- | --- |
| v0.1.x | ✅ Yes |
| < v0.1.0 | ❌ No |
